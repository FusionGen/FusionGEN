Server requirements (important!)

What do I need to run this system?

Folder Permissions

• /application/config/ needs to be writable (see chmod)
• /application/modules/ is writable
• /application/cache/

PHP Version

Recommended PHP Version: 7.1

PHP Extensions

The following PHP extensions are required:
php_mysqli, php_curl, php_openssl, php_soap, php_gd, php_mbstring, php_json
The following modules are missing:
How do I enable PHP extensions?
Go into your PHP directory and find the php.ini file. Mine was located in C:\UwAmp\bin\php\php-7.0.3. Open the file with a text editor and search (CTRL+F) for one of the modules you need to enable. To enable them, simply remove the ; character in front of the line.
Save the file and restart your webserver to apply the changes.

Apache Modules

We recommend FusionGEN in combination with an Apache Webserver.

The Apache modules you'll need are:
mod_rewrite, mod_headers and mod_deflate
The following modules are missing:
How do I enable Apache modules?
Go into your Apache directory and find the httpd.conf file. Mine was located in C:\UwAmp\bin\apache\conf. Open the file with a text editor and search CTRL+F for one of the modules you need to enable. To enable them, simply remove the # character in front of the line.
Save the file and restart your webserver to apply the changes.
